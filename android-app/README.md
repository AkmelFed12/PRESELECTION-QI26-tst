# ASAA Officiel Android (WebView)

This Android Studio project wraps https://asaaofficiel.vercel.app in a WebView.

## Build APK
1. Open Android Studio.
2. File -> Open -> select the `android-app` folder.
3. Wait for Gradle sync.
4. Build -> Build Bundle(s) / APK(s) -> Build APK(s).

## Features included
- App name: ASAA Officiel
- Splash screen (Android 12+)
- Full screen (immersive)
- Offline fallback page
- Adaptive icon

## Change URL
Edit: `android-app/app/src/main/java/com/asaa/app/MainActivity.java`

## Change app name
Edit: `android-app/app/src/main/res/values/strings.xml`

## Change icon colors
Edit: `android-app/app/src/main/res/values/colors.xml`
